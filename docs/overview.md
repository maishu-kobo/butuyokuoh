# プロダクト概要

## 物欲王 (Butuyokuoh) とは

物欲王は、複数のECサイト（Amazon、楽天など）のほしいものリストを一元管理するWebアプリケーションです。

### 解決する課題

- 複数のECサイトに分散したほしいものリストの管理が大変
- 価格変動を追跡したい
- 購入予定の出費を把握したい
- 似た商品を比較して検討したい

## ターゲットユーザー

- 複数のECサイトを利用する個人ユーザー
- 購入計画を立てて買い物をしたい人
- 価格の変動を追跡してお得に買いたい人

## 主要機能

### コア機能

| 機能 | 説明 |
|------|------|
| アイテム登録 | URLを貼るだけで商品情報を自動取得 |
| 手動登録 | URLなしで直接入力も可能 |
| 価格履歴 | Keepa風の価格変動グラフ |
| 出費予定 | 購入予定日別の合計金額計算 |
| カテゴリ管理 | アイテムをカテゴリで分類 |
| 比較グループ | 似た商品をグループ化して比較 |
| 購入済み管理 | 購入したアイテムの履歴保存 |

### 通知・自動化

| 機能 | 説明 |
|------|------|
| 価格下落通知 | Slack/Discord Webhookで通知 |
| 定期価格チェック | 6時間ごとに自動で価格更新 |
| 目標価格通知 | 設定した価格以下になったら通知 |

### インポート・エクスポート

| 機能 | 説明 |
|------|------|
| Webインポート | 公開リストのURLからインポート |
| Chrome拡張機能 | 非公開リストもインポート可能 |
| CSVエクスポート | Excel/スプレッドシート対応 |

### その他

| 機能 | 説明 |
|------|------|
| PWA対応 | スマホのホーム画面に追加可能 |
| 統計ダッシュボード | カテゴリ別/優先度別/価格帯分布 |
| ゴミ箱 | 削除アイテムの復元可能 |

## 対応サイト

### 商品登録（スクレイピング）

| サイト | 対応状況 | 備考 |
|--------|----------|------|
| Amazon.co.jp | ○ | 商品名、価格、画像 |
| 楽天市場 | ○ | 商品名、価格、画像 |
| その他 | △ | OGP情報を取得 |

### ほしいものリストインポート

| サイト | Web版 | Chrome拡張 |
|--------|-------|------------|
| Amazon | 公開のみ | 公開/非公開 |
| 楽天 | 公開のみ | 公開/非公開 |
